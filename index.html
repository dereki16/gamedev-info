Beginning Game Development: An Overview

Game Development is a branch of the Computer Science department. One that doesn't get enough credit because frankly, getting started in this field is an exciting thing. What makes Game Dev so great is there are endless possibilities as to what you can create.

Now it is probably for the best if before you get started, you already have some knowledge about coding promgramming languages like C++, C#, Python, Java etc.

While there may not be a specific language used exclusively for Game Development, C++ is the most popular in Game Dev and most commonly used language for writing game engines. Some examples of popular games made with C++ are Counter-Strike, King Quest, and World of Warcraft. Java is also fairly popular, one popular example being Angry Birds Chrome. Source.
